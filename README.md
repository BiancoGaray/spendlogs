# Expense Manager

[https://sivadass.github.io/expense-manager](https://sivadass.github.io/expense-manager)

Expense manager app build for my own personal use.

### Tech Stack (MERN)

- MongoDB
- Express JS
- React JS
- Node jS

### Frontend

```
cd frontend/
npm run dev
npm run build
```

The app will start at port **3000**

### Backend

```
cd backend/
npm run start
```

The app will start at port **5001**
