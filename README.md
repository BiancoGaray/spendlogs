### Tech Stack (MERN)

- MongoDB
- Express JS
- React JS
- Node jS

### Frontend

```
cd frontend/
npm run dev
npm run build
```

The app will start at port **3000**

### Backend

```
cd backend/
npm run start
```

The app will start at port **5001**
